## Deploy to Netlify

### Option 1: Drag & Drop
1. Add all 10 images to `images/` folder
2. Visit https://app.netlify.com/drop
3. Drag the entire folder
4. Get your live URL!

### Option 2: GitHub
1. Upload to GitHub repository
2. Connect to Netlify
3. Auto-deploys on every push
